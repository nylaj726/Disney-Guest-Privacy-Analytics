## Project Overview
This project demonstrates an Information Technology approach to theme park guest management. It focuses on two core pillars of IT: **Data Analytics** and **Cybersecurity Compliance**.

## Key Features
* **Data Privacy:** Automatically masks sensitive MagicBand IDs using string manipulation to ensure guest anonymity.
* **Operational Analytics:** Calculates real-time demographic data (Average Age) to help park operations understand guest trends.
* **Scalable Logic:** Uses Python dictionaries and loops, making it easy to integrate with larger databases.
